# 五一出行目的地总结

## 一、云南专区

### 1. 云南雨崩

- **时间**：雨崩徒步 + 丽江/大理躺平
- **亮点**：日照金山、神瀑、神湖；"天堂在左，雨崩在右"
- **交通**：丽江机场进出（迪庆机场海拔高，避免高反），公共交通出行
- **行程**：丽江机场 → 飞来寺 → 雨崩 → 香格里拉（独克宗、普达措）→ 迪庆机场
- **Tips**：雨崩即将贯通公路，商业化加速，趁原始赶紧去

### 2. 丽江虎跳峡徒步

- **亮点**：世界级峡谷徒步路线
- **建议**：可结合虎跳峡高路徒步

### 3. 云南腾冲/虎跳峡徒步

- 可搭配：腾冲热海、和顺古镇、芒市

### 4. 丽江/大理躺平

- 适合：古镇漫步、洱海骑行、茶馆发呆

### 5. 丽江泸沽湖

- 适合：放空、慢节奏、湖边发呆

### 6. 丽江+香格里拉（野奢拍照游）✨有详细行程

- **时间**：5月3日-5月7日，共5天
- **特点**：全程野奢民宿、以拍照为主、节奏很松
- **找人要求**：能早起、不怕高反

#### 详细行程

| 天数 | 行程 | 住宿 |
|------|------|------|
| D1（5.3） | 杭州✈️丽江 → 入住棠岸度假酒店 → 营地骑马、纳西扎染 | 棠岸度假酒店 |
| D2（5.4） | 玉龙雪山（大索道+蓝月谷）→ 傍晚转望山雪民宿 → 日照金山+民族旅拍 | 望山雪民宿 |
| D3（5.5） | 高铁去香格里拉 → 入住草原旁心宿 → 喂小动物、躺平适应海拔 | 心宿（草原旁） |
| D4（5.6） | 香格里拉 → 藏服旅拍 → 牦牛火锅 | 心宿 |
| D5（5.7） | 野鹿子咖啡 → 高铁回丽江 → 飞杭州（00:40到萧山） | — |


## 二、新疆专区

### 7. 新疆伊犁环线（详细攻略已给出）

- **时间**：5月3日伊宁集合，5月10日返程，共8天
- **路线**：
  - 5.4 伊宁 → 赛里木湖 → 果子沟大桥 → 伊宁
  - 5.5 伊宁 → 特克斯八卦城
  - 5.6 特克斯 → 琼库什台
  - 5.7 琼库什台深度游玩（骑马、家访）
  - 5.8 琼库什台 → 恰甫其海 → 恰西森林公园 → 库尔德宁镇
  - 5.9 库尔德宁镇 → 伊宁
  - 5.10 自由回程
- **可选景点**：那拉提、库尔德宁、琼库什台、喀拉峻草原 4选n
- **预算**：6-8天核心 + 2天往返 + 1天休整
- **建议**：可加1-2天吐鲁番

## 五、沙漠专区
