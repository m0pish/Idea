# 4.29-5.4 云南行程攻略（昆明→大理→腾冲→芒市）

## 路线概览

```
昆明 →(高铁2h)→ 大理 →(车程5h/飞机1h)→ 腾冲 →(车程2.5h)→ 芒市 →(飞机返程)
```

> 建议：昆明进、芒市出，不走回头路。

---

## 逐日安排

### Day 1｜4.29（周三）— 昆明

| 时间 | 安排 |
|------|------|
| 上午 | 抵达昆明长水机场 |
| 下午 | 滇池海埂大坝、西山龙门（索道上山） |
| 晚上 | 南屏街/昆明老街觅食（过桥米线、小锅米线） |
| 住宿 | 昆明站/南屏街附近 |

---

### Day 2｜4.30（周四）— 昆明→大理

| 时间 | 安排 |
|------|------|
| 上午 | 高铁昆明→大理（约2h，建议8:00左右班次） |
| 中午 | 大理古城安顿+午餐（白族砂锅鱼、饵丝） |
| 下午 | 大理古城漫步、人民路、洋人街、崇圣寺三塔 |
| 晚上 | 古城酒吧街/洱海边散步 |
| 住宿 | 大理古城或才村码头民宿 |

---

### Day 3｜5.1（周五）— 大理环洱海

| 时间 | 安排 |
|------|------|
| 全天 | 环洱海一日（租电动车/包车） |
| 路线 | 才村→喜洲古镇（喜洲粑粑）→双廊→小普陀→挖色 |
| 推荐 | 喜洲稻田、海舌公园、双廊观景台 |
| 晚上 | 返回古城或住双廊 |

---

### Day 4｜5.2（周六）— 大理→腾冲

| 时间 | 安排 |
|------|------|
| 上午 | 大理→腾冲（飞机约50min，或包车约5h） |
| 下午 | 和顺古镇（600年侨乡，比大理古城更安静） |
| 晚上 | 泡温泉（热海温泉或悦椿温泉） |
| 住宿 | 和顺古镇内客栈 |

> 五一期间建议提前买大理→腾冲机票，约200-400元。包车走保腾高速也可以。

---

### Day 5｜5.3（周日）— 腾冲

| 时间 | 安排 |
|------|------|
| 上午 | 热海景区（大滚锅、热气蒸腾） |
| 下午 | 火山地质公园（坐热气球俯瞰火山口）或北海湿地（划草排） |
| 傍晚 | 银杏村（4月底叶子还绿，但村落很美） |
| 晚上 | 腾冲美食：大救驾、稀豆粉、土锅子 |
| 住宿 | 腾冲市区或和顺 |

---

### Day 6｜5.4（周日）— 腾冲→芒市→返程

| 时间 | 安排 |
|------|------|
| 上午 | 腾冲→芒市（车程约2.5h） |
| 中午 | 芒市午餐（傣味手抓饭、撒撇、泡鲁达） |
| 下午 | 勐巴娜西珍奇园、树包塔/塔包树、菩提寺 |
| 晚上 | 芒市德宏芒市机场→返程 |

> 芒市机场航线：直飞昆明（1h）、成都、重庆等。

## 实用提示

1. **五一热门**：大理/腾冲住宿务必提前1-2周预订
2. **防晒**：云南紫外线强，SPF50+必备
3. **温差**：早晚凉（10-15℃），中午热（25-28℃），带薄外套
4. **支付**：古镇小店部分只收现金，备少量
5. **高反**：昆明/大理海拔1900-2000m，一般无感，慢走即可
