# 云南完整攻略：昆明 + 大理 + 丽江 + 腾冲 + 芒市

## 路线总览

```
昆明 →(高铁2h)→ 大理 →(火车/大巴2.5h)→ 丽江 →(泸沽湖往返)→ 丽江
  →(飞机/车回大理)→ 大理 →(飞机/车5h)→ 腾冲 →(车2.5h)→ 芒市 →(飞机返程)
```

> 昆明进、芒市出，不走回头路。全程约 **11-12天**。

---

## 最佳出行时间

| 月份 | 推荐度 | 说明 |
|------|--------|------|
| **3-4月** | ⭐⭐⭐⭐⭐ | 综合最佳，气候舒适、雪山有雪、湖水清透、人少 |
| **9-11月** | ⭐⭐⭐⭐⭐ | 秋高气爽，能见度高，所有景点状态都好 |
| **12-2月** | ⭐⭐⭐⭐ | 雪山最壮观，但早晚冷，腾冲泡温泉最舒服 |
| **5-6月** | ⭐⭐⭐⭐ | 舒服，但雪山雪线可能偏高 |
| **7-8月** | ⭐⭐⭐ | 雨季，云雾多，泸沽湖路况一般 |

### Day 5 — 玉龙雪山远观 + 蓝月谷

> 不上大索道登顶，以远观雪山 + 蓝月谷为主，轻松舒适。

| 时间 | 安排 |
|------|------|
| 8:00 | 出发前往玉龙雪山景区（车程40-60min） |
| 9:00-10:30 | **甘海子**：大草甸开阔平地，正对雪山，是远观拍摄玉龙雪山最经典的位置 |
| 10:30-11:30 | 可选：**云杉坪索道**（小索道，海拔3200m，轻松无高反），森林草甸 + 雪山背景 |
| 12:00-14:30 | **蓝月谷**：沿栈道慢走，玉液湖 → 镜潭湖 → 蓝月湖，晴天中午水色最蓝 |
| 15:00 | 返回丽江 |
| 下午 | 白沙古镇（比丽江更安静，有白沙壁画），远望雪山角度也很好 |
| 晚上 | 丽江休息，准备次日去泸沽湖 |

**不上大索道的好处：**
- 不用抢票（大索道旺季一票难求）
- 没有高反风险
- 节奏轻松，老人小孩都适合
- 甘海子和蓝月谷的雪山视角其实更完整（登顶反而看不到全貌）

**远观雪山最佳拍摄点：**

| 地点 | 特点 |
|------|------|
| **甘海子** | 景区内，草甸+雪山，经典机位 |
| **蓝月谷湖边** | 蓝色湖水+雪山倒影 |
| **白沙古镇** | 古村落+雪山背景，很有层次 |
| **束河古镇** | 屋顶露台望雪山 |

**费用（不上大索道版）：**

| 项目 | 价格 |
|------|------|
| 景区门票（含蓝月谷） | ¥100 |
| 云杉坪索道（可选） | ¥40 |
| 景区环保车 | ¥20 |
| **合计** | **¥120-160** |

> 比上大索道省了 ¥120 索道费，也不用买氧气瓶和租羽绒服。

### Day 6 — 丽江→泸沽湖

| 时间 | 安排 |
|------|------|
| 8:00 | 丽江出发（拼车/包车，约4.5-5h） |
| 12:30 | 途中山路观景台俯瞰泸沽湖全貌（第一眼极震撼） |
| 13:30 | 抵达泸沽湖，入住里格半岛或大落水 |

| 17:30-19:00 | **女神湾看日落** |
| 晚上 | 摩梭篝火晚会（看兴趣），或湖边发呆看星空 |

| 16:30 | 女神湾 | 远眺格姆女神山 |
| 18:00 | 回住处看日落 | — |

### Day 8 — 泸沽湖→丽江→大理

| 时间 | 安排 |
|------|------|
| 6:30 | 最后看一次日出/晨雾 |
| 8:00 | 出发返回丽江（约5h） |
| 13:00 | 抵达丽江，午餐 |

| 时间 | 安排 |
|------|------|

| 下午 | **和顺古镇**（600年侨乡，安静古朴） |

### Day 10 — 腾冲一日

| 时间 | 安排 |
|------|------|
| 上午 | **热海景区**（大滚锅、热气蒸腾） |
| 下午 | **火山地质公园**（热气球俯瞰火山口）或**北海湿地**（划草排） |

## 全程装备清单

- [ ] 身份证（景区实名）
- [ ] 防晒霜 SPF50+、墨镜、遮阳帽
- [ ] 冲锋衣/防风外套 + 薄羽绒（早晚温差大）
