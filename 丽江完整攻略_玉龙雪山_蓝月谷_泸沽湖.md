# 丽江完整攻略：玉龙雪山 + 蓝月谷 + 泸沽湖

## 最佳时间推荐

| 月份 | 推荐指数 | 说明 |
|------|---------|------|
| **3-4月** | ⭐⭐⭐⭐⭐ | 综合最佳，雪山有雪、湖水清透、气温舒适、人少 |
| **9-11月** | ⭐⭐⭐⭐⭐ | 秋高气爽，能见度高，雪山蓝月谷泸沽湖都出片 |
| **12-2月** | ⭐⭐⭐⭐ | 雪山最厚最震撼，但早晚冷（5-15℃），泸沽湖风大 |
| **5-6月** | ⭐⭐⭐⭐ | 舒服，蓝月谷颜色好，但雪山雪线可能偏高 |
| **7-8月** | ⭐⭐⭐ | 雨季，云雾多，雪山常看不全，泸沽湖路况一般 |

> **一句话结论：首选 4月 或 10月，兼顾雪山、蓝月谷、泸沽湖三者最佳状态。**

---

## 推荐行程：5天4晚

```
Day 1  抵达丽江 → 丽江古城
Day 2  玉龙雪山 + 蓝月谷（一日）
Day 3  丽江 → 泸沽湖（半天车程 + 日落）
Day 4  泸沽湖环湖 + 猪槽船
Day 5  泸沽湖日出 → 返回丽江 → 离开
```

## Day 1 — 抵达丽江

| 时间 | 安排 |
|------|------|
| 下午 | 抵达丽江三义机场/火车站，入住古城附近 |
| 傍晚 | 丽江古城漫步：大水车、四方街、狮子山观景台看古城全景 |
| 晚上 | 古城觅食：腊排骨火锅、鸡豆凉粉、丽江粑粑 |
| 住宿 | 丽江古城/束河古镇客栈 |

**Tips：**
- 丽江海拔 2400m，到了别剧烈运动，慢慢适应
- 早睡，第二天要早起上雪山

## Day 2 — 玉龙雪山 + 蓝月谷

> 这是最核心的一天，节奏：**早起抢索道 → 登顶 → 下山蓝月谷 → 傍晚回古城**

### 上午：玉龙雪山

| 时间 | 安排 |
|------|------|
| 6:30 | 出发前往景区（古城出发约40-60min） |
| 7:30 | 抵达景区，排队进入 |
| 8:00-8:30 | 乘坐 **冰川公园大索道**（海拔 4506m → 4680m） |
| 8:30-10:30 | 沿栈道步行至最高观景台（4680m），慢走、拍照、感受雪山 |
| 10:30 | 索道下山 |

**大索道核心提醒：**
- ⚠️ **必须提前1天在"丽江旅游集团"公众号抢票**，旺季一票难求
- 索道上去后走栈道约 300 米到观景台，海拔高，每走几步歇一歇
- 别跑、别跳、别大喊，高反通常是兴奋过后才发作

### 中午：休整

| 时间 | 安排 |
|------|------|
| 11:00-12:00 | 景区内午餐/简单补给，休息调整 |

### 下午：蓝月谷

| 时间 | 安排 |
|------|------|
| 12:30-15:00 | 蓝月谷游览（含在雪山景区内，不用另买票） |
| 重点 | 玉液湖、镜潭湖、蓝月湖，沿栈道慢走约 2-3 公里 |
| 拍照 | **12:00-15:00 晴天时水色最蓝最透**，配合雪山背景极出片 |

| 15:30 | 返回丽江古城 |
| 晚上 | 古城休息，泡酒吧或早睡（第二天去泸沽湖） |

### 玉龙雪山装备清单

| 必带 | 说明 |
|------|------|
| 冲锋衣/厚外套 | 山顶体感可能 0℃ 以下 |
| 薄羽绒/抓绒 | 内层保暖 |
| 墨镜 | 雪地反光强 |
| 防晒霜 SPF50+ | 高原紫外线是平原的数倍 |
| 氧气瓶 1-2 瓶 | 景区内也能买，但贵；提前在古城药店买约 ¥15-30/瓶 |
| 巧克力/能量棒 | 补充热量 |
| 舒适运动鞋 | 栈道不陡但海拔高 |

### 费用参考

| 项目 | 价格 |
|------|------|
| 景区门票（含蓝月谷） | ¥100 |
| 冰川公园大索道 | ¥120 |
| 景区环保车 | ¥20 |
| 古城维护费 | ¥50（部分时段免收） |
| 氧气瓶 | ¥15-30/瓶 |
| 羽绒服租赁（如需） | ¥50-80 |

## Day 3 — 丽江→泸沽湖 + 日落

| 时间 | 安排 |
|------|------|
| 8:00 | 丽江出发前往泸沽湖（拼车/包车/大巴） |
| 12:30-13:00 | 途中经过山路观景台，可停车俯瞰泸沽湖全貌（第一眼非常震撼） |
| 13:30 | 抵达泸沽湖，入住里格半岛或大落水 |
| 下午 | 湖边散步，适应节奏 |
| 17:30-19:00 | **女神湾看日落**（泸沽湖最美日落点之一） |
| 晚上 | 摩梭篝火晚会（看兴趣参加），或湖边安静发呆 |

**交通方式对比：**

| 方式 | 时长 | 价格 | 备注 |
|------|------|------|------|
| 拼车 | 4.5-5h | ¥100-150/人 | 最主流，会在观景台停车 |
| 包车 | 4.5-5h | ¥500-700/车 | 灵活，适合 2-4 人 |
| 大巴 | 5-6h | ¥70-100/人 | 丽江客运站发车 |

**住哪里：**

| 位置 | 特点 | 适合 |
|------|------|------|
| **里格半岛** | 风景最好，安静有氛围 | 第一次去首选 |
| 大落水 | 配套成熟，吃喝方便 | 实用型 |
| 达祖/尼赛 | 最安静 | 二刷或纯休息 |

> ⚠️ 旺季湖景房非常紧俏，建议提前 2 周订。

## Day 4 — 泸沽湖环湖一日

> 这是泸沽湖最精华的一天

### 环湖路线（顺时针）

```
里格半岛 → 尼赛情人树 → 小落水 → 达祖 → 草海·走婚桥 
→ 洛洼码头（坐猪槽船）→ 情人滩 → 女神湾 → 大落水 → 里格
```

环湖一圈约 **75 公里**，推荐包车或租电瓶车。

| 时间 | 安排 | 亮点 |
|------|------|------|
| 7:00 | 里格半岛看日出 | 晨雾中的湖面极美 |
| 8:30 | 出发环湖 | — |
| 9:00 | 尼赛·情人树 | 经典打卡，两棵树立在湖中 |
| 10:00 | 小落水村 | 原生态摩梭村落 |
| 11:00 | **草海 + 走婚桥** | 湿地栈道，芦苇荡中步行 |
| 12:00 | 午餐（草海/达祖附近） | 尝摩梭菜：猪膘肉、酸鱼 |
| 13:30 | **猪槽船游湖** | 在洛洼或大落水乘船，¥50-80/人 |
| 15:00 | 情人滩 | 湖边沙滩，安静发呆 |
| 16:30 | 女神湾 | 远眺格姆女神山 |
| 18:00 | 回里格/大落水看日落 | 不同位置日落各有特色 |
| 晚上 | 湖边星空 | 泸沽湖光污染极少，银河可见 |

**环湖交通对比：**

| 方式 | 价格 | 优缺点 |
|------|------|--------|
| 包车 | ¥300-400/车 | 最轻松，司机兼导游 |
| 租电瓶车 | ¥80-120/天 | 自由，但 75km 电量要确认够用 |
| 拼车环湖团 | ¥100-150/人 | 省心但时间固定 |

## 行前准备清单

### 必备物品
- [ ] 身份证（景区实名）
- [ ] 防晒霜 SPF50+、墨镜、帽子
- [ ] 冲锋衣 + 薄羽绒（雪山必备）
- [ ] 舒适运动鞋
- [ ] 充电宝（泸沽湖充电不一定方便）
- [ ] 少量现金（部分村落不支持手机支付）
- [ ] 氧气瓶（古城药店提前买）
- [ ] 晕车药（去泸沽湖山路多弯）

### 注意事项
1. **雪山索道票提前一天抢**，关注"丽江旅游集团"公众号
2. **泸沽湖住宿旺季提前 2 周订**，尤其湖景房
3. **高原第一天别洗头洗澡太晚**，容易感冒引发高反
4. **去泸沽湖的路弯多**，容易晕车的提前吃药
5. **尊重摩梭文化**，不要随便拍摄当地人

## 时间不够？精简版方案

### 3天2晚（紧凑版）

```
Day 1  抵达丽江 + 古城
Day 2  玉龙雪山 + 蓝月谷
Day 3  束河/白沙 + 离开
```
> 放弃泸沽湖，适合只有周末+1天假的情况

```
Day 1  抵达丽江
Day 2  玉龙雪山 + 蓝月谷
Day 3  泸沽湖（住湖边）
Day 4  泸沽湖半天 → 返回丽江 → 离开
```
> 泸沽湖只住 1 晚，环湖会比较赶

### 5天4晚（推荐版，本攻略）

> 节奏最舒服，三个核心景点都能充分体验
